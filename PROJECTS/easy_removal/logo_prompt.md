# Easy Removal Logo Exploration Prompt

Act as a senior brand identity designer.

Create logo directions for Easy Removal using the approved brand system.

Brand summary:
Easy Removal is a fast, local small-load removal service in Cape Town. It removes garden refuse, household junk and yard or garage clear-outs using a single 4-cubic-metre truck.

The logo must communicate:
- Small-load removal
- Fast response
- Local reliability
- Practical service
- No-nonsense execution

Visual direction:
- Simple vector-based mark
- Compact truck or open load-box concept
- Slight forward motion
- Stable horizontal structure
- Strong readability at small sizes

Colour direction:
- Primary green: #5C8165
- Secondary olive: #86A05C
- Accent ochre: #C39B46
- Charcoal: #202A30
- White: #FFFFFF

Avoid:
- Skip bins
- Large moving trucks
- Furniture moving company styling
- Cartoon rubbish piles
- Heavy industrial graphics
- Hazard stripes
- Overly eco charity cues
- Complex symbols
- Fine detail

Required outputs:
1. Primary horizontal logo
2. Stacked logo
3. Icon-only mark
4. One-colour version
5. Reversed version
6. WhatsApp profile version
7. Truck signage version

Construction rules:
- Use an 8-unit grid
- Slightly rounded corners
- Consistent stroke logic
- Remove internal detail at small sizes
- Keep the wordmark strong and legible

Generate 3 distinct logo directions:
1. Truck-led direction
2. Load-box direction
3. Cleared-space direction

For each direction, explain:
- Concept
- Construction logic
- Why it fits the brand
- Where it may fail
- Best usage context
