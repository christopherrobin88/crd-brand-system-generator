# CRD Brand System Generator

A structured system for building commercially viable brand identities.

This is not a logo generator.  
This is a repeatable framework for:

- Brand strategy
- System design
- Visual direction
- Figma-ready output
- Scalable automation

---

## Structure

/CRD_SYSTEMS  
  /brand_generator  
    - prompt.md  
    - process.md  
    - model_roles.md  

  /quality_gate  
    - checklist.md  

---

## How to Use

1. Fill input block (prompt.md)
2. Run through system (process.md)
3. Assign correct models (model_roles.md)
4. Validate output (quality_gate/checklist.md)
5. Build in Figma

---

## Principle

System first.  
Visuals second.  
Execution always manual.
